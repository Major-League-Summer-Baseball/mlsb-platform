Parameter
Field Type Description
year int The year of the games
league_id int The Leagues-ID

Field Type Description
game_id int The Games-ID
away_bats/td> List of Bats Contains all the bats for the away Team
away_score int The number of runs scored by the away Team
away_team Team The away Team
home_bats/td> List of Bats Contains all the bats for the home Team
home_score int The number of runs scored by the home Team
home_team Team The home Team
league League The league the game was part off
