Response Object

A response Object is an object all API use to respond to requests.

The following table shows all attributes, what type each attribute is, and describes what is does:

Attribute Type Description
success boolean A boolean that tells if the request was successful
message string An error message
failures list A list of failures that have occurred
data list or Object The requested data
object_id int Post request will return the id created for the object (e.g. Put Player->player_id:int
