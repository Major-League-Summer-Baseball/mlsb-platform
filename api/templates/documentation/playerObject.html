Player Object

A player Object is the information stored relating to the player and its attributes.

The following table shows all player attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
player_id integer Generated The player's identifier -- --
player_name string yes The player's name -- --
gender string yes The player gender -- F, M, or T
email string yes The player's email -- must be a unique email
