League Event - View

GET

{{route.vleagueevents}}/<:year>

Permission: none

Parameter
Field Type Description
year int Specify the year for the levents.
Success 200 returns a list of the following League Event Dates
Field Type Description
league_event_date_id int The id of the league event date
league_event_id int The id of the league event
description str The html description of the league event
name str The name of the league event
active boolean Whether the league event is active
date str The date of the event (%Y-%M-%D) or TBD
time str The time of the event (HH:MM) or TBD
attendance str How many players have attended or planning to attending
