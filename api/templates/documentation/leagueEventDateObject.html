League Event Date Object

A LeagueEventDateObject is the information stored relating to a League Event Date and its attributes.

The following table shows all League Event Date attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
league_event_date_id int Generated The league event's identifier -- --
league_event_id int yes The league event that the date is for -- --
date string yes The date of the event -- Format: %YYYY-%MM-%DD
time string yes The start time of the event -- Format: %H:%M
