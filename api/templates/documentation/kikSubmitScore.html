Kik - Captain submit scores

Submit scores for a games that has been played. To see games that needs scores see captain games

POST

{{route.kiksubmitscore}}/

Permission: Kik

Parameter
Field Type Description
game_id int The game's id
kik str The captain's kik user name
score int The number of runs scored by team
hr list of int The player's id who hit homeruns
ss list of int The player's id who hit special singles
Responses
Status Code Response Object Status
200 -- True
436 -- Game does not exist
404 -- Captains kik name was not found
403 -- Captain not a captain of either team for the game
400 -- Bad request: read message for more details
432 -- Invalid Field: player cannot hit a special single
