Sponsor Object

A sponsor Object is the information stored relating to a sponsor and its attributes.

The following table shows all sponsor attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
sponsor_id int Generated The sponsor's identifier -- --
sponsor_name string yes The name of the sponsor -- --
link string no A link to the sponsor -- --
description string no A description of the sponsor -- --
