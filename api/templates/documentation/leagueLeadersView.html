League Leaders - View

POST

{{route.vleagueleaders}}

Permission: none

Parameter
Field Type Description
group_by_team int If 1 then all stats for players are grouped by team (players only show up for each team they are one). If false Otherwise all plThe hit classification to find league leader
stat str The hit classification to find league leader
year int Specify the year for the leaders, leaders are for entire league history. If no year is specified then gets leaders over the whole league history.
Success 200 returns a list of the following
Field Type Description
hits float The number of the hits that player had (based on the parameter given)
id int The players id
name str The name of the player
team str The team name the player was playing for
Example
