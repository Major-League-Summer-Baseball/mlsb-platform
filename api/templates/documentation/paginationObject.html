Pagination Object

A Pagination Object is used for any GET request that returns a list of Objects

The following table shows all bat attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Description
has_next boolean True if there are more pages
has_prev boolean True if there are previous pages
items Array An array of the objects requested
next_url String The next page url, None if there are no more pages
prev_url String The previous page url, None if there are previous pages
total int The total number of items for all pages
pages int The totla number of pages 0 --
inning int The inning the bat occurred in
