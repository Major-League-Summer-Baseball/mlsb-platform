Kik - Player Subscribe to Team

Allows a player to subscribe to a team

POST

{{route.kiksubscribe}}/

Permission: Kik

Parameter
Field Type Description
kik str The kik user name
team int The team's id
name str The name of the player
Responses
Status Code Response Object Status
200 -- True
434 -- Team does not Exist
441 -- Player not on team
