Team Object

A team Object is the information stored relating to the team and its attributes.

The following table shows all team attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
team_id integer Generated The team's identifier -- --
team_name string Combination of Sponsor & Color The team's name -- --
captain int no The player object of the team's captain -- --
sponsor_id int no The sponsor id of the sponsor of the team -- --
color string no The color of the team -- --
year int yes The year of the team Current year --
espys int no The total espys points of the team 0 --
