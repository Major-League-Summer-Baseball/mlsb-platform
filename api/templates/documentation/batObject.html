Bat Object

A bat Object is the information stored relating to bat event and its attributes.

The following table shows all bat attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
bat_id int Generated The bat's identifier -- --
game_id int yes The game's identifier -- --
team string Comes from team_id The team's name -- --
team_id int yes The team's identifier -- --
player string Comes from player_id The player's name -- --
player_id int yes The player's identifier -- --
rbi int no The number of runs batted in 0 --
inning int no The inning the bat occurred in -- --
hit string yes The hit classification -- 'S' = Single
'SS' = Sage Single
'D' = Double
'HR' = Homerun
'K' = strikeout
'E' = error
'FC' = fielder's choice
'FO' = fly-out
'GO' = ground out
