Game Object

A game Object is the information stored relating to a game and its attributes.

The following table shows all game attributes, what type each attribute is, and any restrictions on that attribute:

Attribute Type Required Description Default Restrictions
home_team_id integer no The home team's identifier -- --
away_team_id integer no The away team's identifier -- --
date string yes The date of the game -- Format: %YYYY-%MM-%DD
time string yes The start time of the game -- Format: %H:%M
league integer yes The league's identifier -- --
game_id integer yes The game's identifier -- --
